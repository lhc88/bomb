### 🔴 우선순위 1: 연쇄 폭발 미동작

**상태**: ✅ 수정 완료

**파일**: `lib/game/bomberman_game.dart` line 155-203

**문제 원인**:
- 폭발 범위 내의 다른 폭탄을 순회하지 않았음
- 다른 폭탄의 explode() 메서드를 호출하지 않았음

**수정 내용**:
```dart
// ✅ 추가된 코드
final bombsToExplode = <BombComponent>[];
for (final otherBomb in bombs) {
  if (otherBomb != bomb &&
      explosion.isAffected(otherBomb.gridX, otherBomb.gridY)) {
    bombsToExplode.add(otherBomb);
  }
}

for (final chainBomb in bombsToExplode) {
  if (!chainBomb.hasExploded) {
    chainBomb.explode();  // 연쇄 폭발!
  }
}
```

**테스트 방법**:
1. 여러 폭탄을 인접하게 배치 (1칸 간격)
2. 하나의 폭탄 터뜨리기
3. **예상 결과**: 다른 폭탄들도 함께 폭발 ✅

**수정 결과**: ✅ 정상 동작
